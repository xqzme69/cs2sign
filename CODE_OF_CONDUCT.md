# Code of Conduct

This project is maintained as a practical engineering space. Keep discussions
focused, technical, and respectful.

## Expected Behavior

- Be direct without personal attacks.
- Keep reports reproducible and relevant to `cs2sign`.
- Respect maintainers' time by reading existing documentation before opening
  duplicate issues.
- Assume mistakes can be fixed, but do not tolerate harassment or abuse.

## Unacceptable Behavior

- Personal insults, threats, harassment, or discriminatory language.
- Posting private information without permission.
- Spamming issues, pull requests, or discussions.
- Demanding support for unrelated projects.

## Enforcement

Maintainers may edit, hide, lock, or delete comments and issues that violate
this code. Repeated or severe violations may result in a block from the
project.
